## 📖 介绍

因没找到合适的react多页应用模板，手动搭建了一个。

## 🏠 主页

[在线预览](http://react-multiple.lgh930.com/)

## 💻 使用

```
下载：git clone https://github.com/liuguanhua/react-multiple-template.git

开发：
    cd react-multiple-template
    yarn install
    yarn start
    http://localhost:3000

生产：
    yarn build
    cd build
    yarn global add serve
    serve
    http://localhost:5000
```
